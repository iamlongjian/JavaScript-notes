### 对象
##### 对象是一种复合的数据类型，可以在一个对象中保存多个不同数据类型的属性。
##### 对象的分类
- 内建对象:ES标准中定义的对象，如：Math，String，Number，Function...
- 宿主对象:Js运行环境中提供的对象，如：DOM的document.write(); BOM的console.log();
- 自建对象:由开发人员自己创建的对象。
```
语法
var user = {
    name:'哈皮',
    sex:'男',
    age:18
}
console.log(user.name);             //哈皮
```
