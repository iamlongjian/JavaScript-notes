### 基本数据类型和引用数据类型
- 基本数据类型的值相互不受影响`var a=10,b=a;a++ //结果为a=11,b=10`比较的是**值**
- 引用数据类型是相互受影响的,比较的是**内存地址**
```
var obj = new object();
obj.name = 'long';
var obj2 = obj;
obj2.name = 'hhh';
console.log(obj.name);              //结果为hhh
```

### 对象字面量
-使用字面量创建对象
- 属性名和属性值之间使用':' 
- 多个名值对之间用','隔开 **如果**是最后一条属性,后面不要加','
```
var user = {
    name : '猪八戒',
    sex  : '男',
    age  : 2000,                            //正常属性名可以不加 ""
    "1321@#@#$@" : 'hello'                 //如果属性名比较奇怪,请加上 ""
}
```
