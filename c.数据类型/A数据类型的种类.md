
### 数据类型
- `Number` 数字类型
123 **整形**
1.2 **浮点型**
-1  **负数**
1.2e4 **12000**
NaN **not a number(少用)**
Infinity **无穷大(少用)**
------------
- `string` 字符串
'longer'
"longer"
'longer:"123"' **单引号包双引号或者双引号包单引号**
'he\'s'  **\为转义字符，字符串中间的单引号不和外面的引号冲突**
`
啊~  祖国   **字符串换行tab的上面一个键**
`
------------
- `Boolean` 布尔值
true 真
false 假
------------
- `Array`    数组
['abc','a',1,2,ture,{a:b}] **两个[]，中间可以是字符串，数字，布尔值，对象**
------------
- `Object` 对象
```
定义o对象，里面含有a(值为1),b(值为2)
var o = {
    a:1,
    b:2
};
定义一个变量X,X的值为 a+b
var x = o.a + o.b;
输出X的结果为3
document.write(x);
```
